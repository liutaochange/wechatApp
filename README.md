
仿网易云音乐APP的微信小程序

接口统一使用网易云新版API 具体参考 https://github.com/darknessomi/musicbox/wiki/

###目前实现功能

1. 用户
2. 歌单
3. FM
4. 播放
5. 评论
6. MV
7. 专辑
8. 歌手
9. 登录
10. 歌曲红心,FM trash，收藏单曲至歌单
11. 收听记录
12. 歌单歌曲推荐
13. 迷你播放条
14. 电台，节目
15. 搜索
